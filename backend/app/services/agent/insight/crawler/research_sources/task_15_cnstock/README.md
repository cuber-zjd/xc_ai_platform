# 中国证券网新闻资讯（头条资讯检索代理）爬虫验证模块

由于中国证券网上证搜索域名发生 301/302 重定向到其官网首页，不向免登录公众开放。本模块在免登录前提下，利用今日头条搜索平台资讯频道代理检索，并通过强力过滤作者字段（必须包含 `"中国证券网"` 或 `"上海证券报"`）和跳转 URL 二次解码还原手段，精准提取中国证券网的原创报道。

## 操作指南
运行以下命令执行验证：
```bash
python test_cnstock.py
```
数据将自动降序去重后，保存在 `data/mixue_news.json` 和 `data/chabaidao_news.json` 中。
