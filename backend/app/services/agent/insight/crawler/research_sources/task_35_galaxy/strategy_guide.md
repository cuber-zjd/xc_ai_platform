# 银河证券爬虫安全与防封策略

- **后台 API 直连**：避开前台没有检索能力的静态页面，直接利用抓包逆向所得的官方拉取接口：`website2020/doc/queryDocList`。
- **支持官方关键字**：该 API 支持传入 `keyword` 检索参数进行服务器端过滤。
- **无数据安全保护**：针对接口在无匹配结果时返回 `data: null` 的特性，特别添加 `(res_data.get("data") or {})` 的 Python 安全节点保护，防止抛出 None 异常。
- **超链接拼接**：基于 `docId` 完美还原研报阅读详情链接。
- **多词冷却**：两次检索切换时强制挂起 5 秒。
