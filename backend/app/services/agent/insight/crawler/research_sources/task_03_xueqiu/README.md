# 任务一：雪球网未登录资讯爬取

本任务旨在利用 **Playwright** 抓取雪球网中关于“蜜雪冰城”和“茶百道”的最新资讯。

## 目录结构说明

- `README.md`: 任务说明（当前文件）
- `test_xueqiu.py`: 主要爬取验证脚本
- `data/`: 抓取结果数据存放目录
  - `mixue_news.json`: 蜜雪冰城相关最新资讯
  - `chabaidao_news.json`: 茶百道相关最新资讯

## 技术方案

1. 使用 `playwright` 启动 Chromium。
2. 配置抹除自动化指纹，避免直接触发安全盾拦截。
3. 打开雪球首页，确保其下发必要的 Cookie 凭证。
4. 调用雪球搜索接口（或通过 UI 触发搜索并监听 Response），获取相应的资讯数据。
5. 过滤出帖子的主要信息，包括标题、作者、创建时间、内容与链接，并持久化到本地 `data/` 目录中。
