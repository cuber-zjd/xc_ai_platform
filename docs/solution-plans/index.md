# 方案文档中心

AI Platform 的平台级、子系统级、业务流程级方案统一沉淀在本目录。查询项目方案、开发计划、阶段进度时，优先阅读本目录下的 Markdown 文件；如同时存在 HTML 和 Markdown，优先阅读 Markdown 版。

## 目录约定

- 方案按业务域分目录归档，例如 `帆软报表助手/`、`市场洞察/`。
- 新增方案文件必须使用中文命名，避免继续新增英文短横线文件名。
- 同一业务域的计划、验收、待办、截图和归档材料应放在同一个业务目录下。
- 新增或移动方案后，需要同步更新本索引。
